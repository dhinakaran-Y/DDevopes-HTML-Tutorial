HTML Tags Intro.

The tag is a piece of markup that define an element on the page.It tells to the browser what kind of content is given, like ia heading, paragraph, medias, meta dat, form, inputs, table, list etc...

We are write two types of tags container tags & self closing tags

The container tags (or) paired tags had 3 parts Start Tag(<h1>), Element Content , Close Tag(</h1>)

And the self closing tags are like (<br>), (<hr>), (<img src='./path'>)

Attributes Intro.

All HTML elements can have attributes.The attributes provide additional information about HTML elements.The attributes must specified in the start tag

Usually the attributes structure like name/value pairs: attribute name = "value"

Meta Tags.

The meta tags are give metadata of the html document.

It mentioned into the header only.

To responsive design

Form Tags.

In real world a paper of form a person fills the information in the form to give any information to any person or a private or public corporation.

Same as in webpage the form tags are used to interactive with the user and the browser to send and receive the data.

Form had many types of input tags thats are used to get the data.

Below example shows a simple HTML form.

        <form action="/submit" method="post">
            <!-- Text input for username -->
            <label for="username">Username:</label>
            <input type="text" id="username" name="username" required>
        
            <!-- Password input -->
            <label for="password">Password:</label>
            <input type="password" id="password" name="password" required>
            <br><br>
        
            <!-- Radio buttons for gender -->
            <label>Gender:</label>
            <input type="radio" id="male" name="gender" value="male">
            <label for="male">Male</label>
            <input type="radio" id="female" name="gender" value="female">
            <label for="female">Female</label>
            <br><br>
        
            <!-- Submit button -->
            <input type="submit" value="Submit">
        </form>
